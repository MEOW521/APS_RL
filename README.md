## 问题描述

本项目面向汽车制造场景中（冲、焊、涂、总）的 **APS多约束排产问题**，采用当前主流的强化学习进行求解

在真实生产环境中，排产结果需要考虑多种工艺和生产约束，如：
- 相同车型连续生产数量有上限
- 相同白车身连续生产数量有上限
- 相同整车物料号连续生产数量有上限
- 单车型下白车身生产种类有上限
- 单白车身下物料号生产种类有上限

此外还有许多其他的约束条件：拆批、插单、指定班次生产等；此项目暂不考虑除上述5种约束外的其他约束

排产目标：
- 在尽量不违反约束条件的情况下，完成订单排产。

主要技术：
- PPO-Lagrangian
